# stock-prediction

本项目尝试使用几种经典的神经网络实现对A股上证50的预测

## 网络模型

目前已实现的网络模型有

- [X] DNN 深度神经网络 
- [X] LSTM 长短期神经网络

## 机器学习框架

目前已实现的机器学习框架有

- [X] 百度自研PaddlePaddle框架, [项目说明](./paddle.md)
- [ ] Tensorflow 2, 开发中
